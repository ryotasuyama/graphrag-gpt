| 1 | `BracketShapeType` ノードを検索するクエリが存在しない | **完全未対応** |

### 2. `search_bracket_types(question: str) -> dict` を新規追加

**位置**: `search_api_methods()` の直後に追加。

**処理内容**:

1. `is_bracket_task(question)` を呼び出す。`False` の場合は `{"bracket_types": [], "dimension_types": []}` を返す。
2. Neo4j から `BracketShapeType` ノードを全件取得する。

   ```cypher
   MATCH (b:BracketShapeType)
   RETURN b.bracket_type  AS bracket_type,
          b.shape_name    AS shape_name,
          b.bracket_params AS bracket_params,
          b.category      AS category,
          b.usage_note    AS usage_note
   ORDER BY b.bracket_type
   ```

3. Neo4j から `DimensionType` ノードを全件取得する。

   ```cypher
   MATCH (d:DimensionType)
   RETURN d.dim_type   AS dim_type,
          d.shape_name AS shape_name,
          d.params     AS params,
          d.usage_note AS usage_note
   ORDER BY d.dim_type
   ```

4. 両クエリの結果を `{"bracket_types": [...], "dimension_types": [...]}` の辞書にまとめて返す。
5. 例外発生時は `{"bracket_types": [], "dimension_types": []}` を返す（安全）。

### 4. `search_help_pages()` を修正 ― DESCRIBES 順方向の活用

**現在**: HelpPage ノードを取得するのみ。

**修正後**: 取得した HelpPage が `DESCRIBES` リレーションで指す Method / Object の名前も一緒に取得する。

変更する Cypher クエリ（主検索・フォールバック両方）:

```cypher
-- 修正後（主検索）
MATCH (h:HelpPage)
WHERE any(kw IN h.keywords WHERE toLower(kw) CONTAINS toLower($term))
   OR toLower(h.title) CONTAINS toLower($term)
OPTIONAL MATCH (h)-[:DESCRIBES]->(target)
RETURN h.id AS id, h.title AS title, h.summary AS summary,
       h.raw_text AS raw_text, h.page_type AS page_type,
       collect(target.name) AS describes_targets
LIMIT 5
```

戻り値リストの各辞書に `"describes_targets": [...]` フィールドが追加される。

`_build_context()` を修正し、`describes_targets` が存在する場合は以下のように表示する:

```
### [1] ブラケット
概要: ブラケットを作成します。
関連API: ShipBracket, CreateBracket
本文抜粋: ...
```

### 5. `search_api_methods()` を修正 ― DESCRIBES 逆方向の活用

**現在**: Method ノードとその Parameter / ReturnValue を取得するのみ。

**修正後**: 取得した Method を `DESCRIBES` する HelpPage の id・title も一緒に取得する。

既存の Method 取得クエリに以下を追加:

```cypher
OPTIONAL MATCH (hp:HelpPage)-[:DESCRIBES]->(m)
RETURN m.name AS name, m.description AS desc,
       obj.name AS obj_name,
       rv.description AS rv_desc, rt.name AS rv_type,
       collect(hp.id) AS help_page_ids,
       collect(hp.title) AS help_page_titles
LIMIT 5
```

`_build_api_context()` を修正し、`help_page_ids` が存在する場合は以下のように表示する:

```
### [API] CreateBracket（オブジェクト: ShipDesign）
説明: ブラケットを作成します。
関連ヘルプ: cmd_ship_bracket（ブラケット）
引数: ...
戻り値: ...
```
